# Expo Stopwatch App

A simple stopwatch app using Expo & React Native.  
Start, stop, and reset a timer with ease.

## ⏱️ Features

- ▶️ Start timer
- ⏸️ Stop/Pause timer
- 🔄 Reset timer
- 🕓 Display elapsed time in HH:MM:SS

## 📦 Technologies Used

- [Expo](https://expo.dev/)
- [React Native](https://reactnative.dev/)
- React Hooks (`useState`, `useRef`, `useEffect`)
